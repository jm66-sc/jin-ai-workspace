# 共创资料池

> 麦龙 + 金龙 采集的所有内容存放在这里，供 Boss + Claude 双认知批注后入库 iMA。

## 文件夹说明

| 文件夹 | 用途 | 谁来写 |
|--------|------|--------|
| `原文存档/` | 原始内容（文章全文 / 视频转文字稿） | 麦龙 + 金龙 |
| `中心思想卡片/` | 每篇原文的摘要卡片 + Boss认知 + Claude认知 | 麦龙生成，Boss+Claude批注，金龙确认完整后入库 iMA |
| `落地讨论/` | 某次讨论后的共识记录 → 行动项 | Claude + Boss 共同整理 |

## 命名规范

```
原文存档/YYYY-MM/序号-来源简称-标题关键词.md
中心思想卡片/YYYY-MM/卡片-序号-标题关键词.md

```

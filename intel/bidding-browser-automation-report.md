## 一、背景

Boss 要求对国企招标平台进行自动化监控，重点关注以下关键词：

> 消防设备、消防分包、消防劳务、机电设备、楼控设备、系统集成、PLC、变频器、自控、仪表

监控目标：

- 语义解析后与关键词匹配，过滤出高相关度条目

## 二、技术路径验证结果

### ✅ 平台1：成建e采（cjebuy.com）

| 项目 | 结果 |
|------|------|
| 列表页 | 无需登录，可直接访问 ✅ |
| 详情页 | navigate 直连 URL 即可访问 ✅ |
| 正文内容 | 可渲染（动态加载 JS，需等页面稳定）✅ |

**URL 规律：**
```
列表页：  https://www.cjebuy.com/portal/list.do?chnlcode=tender
详情页：  https://www.cjebuy.com/portal/detail.do?docid={32位hex}&chnlcode=tender&objtype=
```

**docid 来源：** 从列表页 `<a href>` 或 `<button onclick>` 中提取（32位十六进制字符串）。

**已验证详情页字段：**
- 标题 ✅
- 报名截止时间 ✅
- 联系人、联系电话 ✅
- 正文内容（正文内容区可见，部分段落动态渲染）✅

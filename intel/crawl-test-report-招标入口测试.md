## 三、政府采购网（ccgp.gov.cn）问题

**症状：** 搜索结果页返回"访问过于频繁"

```
您的访问过于频繁,请稍后再试。 事件ID: dc001d7eaa177544197631a0f250
访问时间: 2026/04/05 19:19:36
```

**原因：** ccgp.gov.cn 有严格的 IP 频率限制，同一 IP 短时间多次请求会被封

**解决方向：**
1. **加请求间隔**（每次请求间隔 ≥ 30秒）
2. **代理池轮换**（多个 IP 交替使用）
3. **缓存结果**（已抓过的页面不重复抓）
4. **用 search.ccgp.gov.cn 代替**（需降低请求频率）
5. **金龙本地抓取**（本地 IP 可能不在限制范围内）

## 六、技术路径说明

### 为什么直接 curl 抓得到？
这些平台（成建e采、善建云采、蜀道集采）都是**传统 JSP/PHP/静态HTML页面**，不是 SPA，内容直接嵌入在 HTML 里。直接 `curl` 即可获取，不需要 browser 自动化。

### 关键技术参数
```bash
curl -s -L --max-time 8 \
  "https://zb.shudaojt.com/zbgg/zhaobiao.html" \
  -H "User-Agent: Mozilla/5.0" \
  | grep -oE 'href="/zbgg/[^"]*"'
```

### 关键词过滤（适用于所有平台）
```
消防设备、消防分包、消防劳务、机电设备、楼控设备、
系统集成、PLC、变频器、自控、仪表、变压器、
配电柜、电气、弱电、智能化
```

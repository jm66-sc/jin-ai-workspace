SmartScout 桌面启动器
======================

已在您的桌面创建了两个启动器：

1. SmartScout.app (应用程序) - ★推荐★
   - 双击即可启动系统
   - 自动检测并安装缺失的依赖
   - 会自动打开终端窗口并运行服务
   - 启动完成后自动打开浏览器

2. Start-SmartScout.command (终端脚本)
   - 双击在终端中运行
   - 提供菜单选择启动选项
   - 自动检测依赖并提供安装选项

📦 智能依赖安装
---------------
启动器会自动检测系统是否已安装必要的依赖：
• Python 3 - 后端运行环境
• Node.js - 前端运行环境
• npm - Node.js 包管理器

如果缺少依赖，启动器会：
1. 提示您缺少的依赖
2. 提供"自动安装"选项（推荐）
3. 提供"手动安装指导"
4. 帮助您完成安装过程

🚀 使用方法
---------
1. 双击桌面上的 SmartScout.app
2. 如有需要，按照提示安装依赖
3. 系统会在终端中启动所有服务
4. 启动完成后会自动打开浏览器访问 http://localhost:3001
5. 按 Ctrl+C 停止所有服务

🔧 常见问题
---------
Q: 提示"请先安装 node"怎么办？
A: 选择"自动安装"选项，系统会帮您安装 Node.js 和 npm。

Q: 安装需要管理员权限吗？
A: macOS 自动安装使用 Homebrew，可能需要输入密码。

Q: 之前安装过 Python/Node.js，为什么还要安装？
A: 系统检测的是命令行工具，如果您安装了但不在 PATH 中，可能需要重新安装。

Q: 可以跳过依赖检查吗？
A: 不建议，缺少依赖会导致系统无法运行。

🎯 新功能体验
-----------
启动后可以在前端界面体验：
1. 系统控制面板（顶部可折叠面板）- 控制启动/停止系统
2. 完整12字段数据展示（第4步数据验证）- 查看和打印完整数据
3. 状态反馈滚动显示（实时消息）- 监控系统运行状态

📁 日志文件
---------
如有问题，请检查项目目录下的 logs/ 目录：
• backend.log - 后端服务日志
• frontend.log - 前端服务日志

💡 提示：首次运行建议使用 SmartScout.app，它会自动处理所有依赖安装。

## 实现的5个核心API端点
1. **规则确诊接口** `POST /api/rule-diagnosis`
   - 功能：基于URL列表进行规则确诊，调用DeepSeek规则扩展器
   - 请求体：URL列表、初始黑白名单
   - 响应：推荐规则、项目ID、状态

2. **保存规则接口** `POST /api/rules/{project_id}`
   - 功能：保存人工确认后的规则到数据库
   - 请求体：黑白名单、人工确认标志
   - 响应：成功状态、规则数量

3. **启动生产接口** `POST /api/production/{project_id}/start`
   - 功能：启动爬虫生产任务，集成现有生产者+消费者流水线
   - 请求体：目标数量、并发数、最大页数
   - 响应：任务ID、状态、预计时间

4. **获取结果接口** `GET /api/results/{project_id}`
   - 功能：实时获取提取结果，支持分页和排序
   - 查询参数：limit、offset、order_by
   - 响应：结果列表、总数、项目状态

5. **提交反馈接口** `POST /api/feedback`
   - 功能：提交人工验证反馈
   - 请求体：结果ID、准确度评分、反馈文本、建议字段
   - 响应：反馈ID

**附加接口**：
6. **任务状态查询接口** `GET /api/tasks/{task_id}`
   - 功能：查询生产任务状态和进度

## 数据库扩展
在现有SQLite数据库基础上新增了两个表：

### tasks表（任务管理）
```sql
CREATE TABLE tasks (
    task_id TEXT PRIMARY KEY,
    project_key TEXT NOT NULL,
    status TEXT DEFAULT 'pending',
    target_count INTEGER,
    processed_count INTEGER DEFAULT 0,
    successful_count INTEGER DEFAULT 0,
    skipped_count INTEGER DEFAULT 0,
    started_at TIMESTAMP,
    completed_at TIMESTAMP,
    error_message TEXT,
    FOREIGN KEY (project_key) REFERENCES projects (url_key)
)
```

### feedback表（用户反馈）
```sql
CREATE TABLE feedback (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    result_id INTEGER NOT NULL,
    accuracy_rating INTEGER CHECK (accuracy_rating BETWEEN 1 AND 5),
    feedback_text TEXT,
    suggested_fields JSON,
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (result_id) REFERENCES results (id)
)
```

### 2. 启动生产接口实现（main.py:359-413）
```python
@app.post("/api/production/{project_id}/start", response_model=StartProductionResponse)
async def start_production(project_id: str, request: StartProductionRequest, background_tasks: BackgroundTasks):
    """启动生产接口，根据目标产值启动爬虫生产"""
    logger.info(f"启动生产请求: {project_id}, 目标: {request.target_count}")

    try:
        # 验证项目存在
        with SQLiteManager() as db:
            project = db.get_project(project_id)
            if not project:
                raise HTTPException(status_code=404, detail="项目不存在")

        # 生成任务ID并创建任务记录
        task_id = f"task_{uuid.uuid4().hex[:8]}"
        conn = sqlite3.connect(DB_PATH)
        cursor = conn.cursor()
        cursor.execute(
            "INSERT INTO tasks (task_id, project_key, status, target_count) VALUES (?, ?, 'pending', ?)",
            (task_id, project_id, request.target_count)
        )
        conn.commit()

        # 在后台运行生产任务
        background_tasks.add_task(
            run_production_task,
            task_id, project_id, request.target_count, request.concurrency, request.max_pages
        )

        return StartProductionResponse(
            task_id=task_id,
            project_id=project_id,
            status="started",
            estimated_time="30分钟"
        )
```

# SmartScout 快速开始指南

## 三步启动

### 第一步：配置 API Key

1. 打开文件夹中的 `config/secrets.yaml`
2. 找到这一行：
   ```
   api_key: "YOUR_API_KEY_HERE"
   ```
3. 把 `YOUR_API_KEY_HERE` 替换成你的 DeepSeek API Key

**如何获取 API Key？**
- 访问 https://platform.deepseek.com/api_keys
- 登录你的 DeepSeek 账号
- 点击"创建新密钥"
- 复制密钥内容到 `secrets.yaml`

### 第二步：启动服务

**Mac 用户：**
- 双击 `start.sh` 文件
- 或在终端运行：`./start.sh`

**Windows 用户：**
- 双击 `start.bat` 文件
- 或在 PowerShell 运行：`.\start.bat`

**Linux 用户：**
- 在终端运行：`bash start.sh`

### 第三步：打开浏览器

启动脚本会自动打开浏览器，访问：
```
http://localhost:3001
```

如果没有自动打开，手动访问上面的地址。

## 使用流程

### 1. 规则配置
- 输入目标 URL（政府采购网搜索页面）
- 系统自动抓取前50条样本
- DeepSeek 分析并推荐黑白名单关键词
- 你确认或修改规则

### 2. 生产配置
- 设置目标数量（要抓多少条）
- 设置最大页数（最多翻几页）
- 点击"启动生产"

### 3. 数据验证
- 实时查看抓取进度
- 查看提取的字段
- 点击"原文链接"验证准确性

## 常见问题

### Q: 启动时报错"API Key 无效"
**A:** 检查 `config/secrets.yaml` 中的 API Key 是否正确填写

### Q: 浏览器打不开
**A:** 手动访问 http://localhost:3001

### Q: 抓取速度很慢
**A:** 这是正常的，政府网站有反爬机制，需要等待

### Q: 数据怎么导出
**A:** 在数据验证页面，点击"导出 CSV"按钮

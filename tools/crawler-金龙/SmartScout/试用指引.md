# SmartScout 桌面试用指引
2026-03-18 下午收口版本

## 🚀 一分钟快速开始

### 第一步：安装Python依赖
```bash
# 进入项目目录
cd /path/to/SmartScout

# 安装Python依赖包
pip install -r requirements_api.txt
```

### 第二步：一键启动
```bash
# 启动后端服务 + 前端界面
python run.py
```

### 第三步：打开浏览器
访问 **http://localhost:8000** 即可使用完整系统。

---

## 📁 文件夹结构说明
```
SmartScout/
├── run.py              # 一键启动脚本（包含环境检查）
├── main.py             # FastAPI后端 + 前端静态文件托管
├── frontend/dist/      # 前端构建文件（已包含）
├── config/             # 配置文件
│   ├── settings.yaml   # 公开配置
│   └── secrets.yaml    # 敏感配置（已包含API密钥，请勿泄露）
├── src/                # 核心爬虫代码
├── data/               # 数据库和临时文件
└── requirements_api.txt # Python依赖清单
```

## ⚙️ 配置说明
- **前端已集成**：无需单独启动，后端自动托管静态文件
- **API密钥已配置**：`config/secrets.yaml` 中已包含测试用的DeepSeek API密钥
- **端口8000**：前端和后端共用同一个端口（避免3000端口冲突）

## 🔧 已知问题与注意事项
1. **API密钥安全**：`secrets.yaml` 包含真实API密钥和代理凭证，试用时请注意保护
2. **依赖检查**：首次运行 `python run.py` 会检查Python版本和依赖包，如有缺失请手动安装
3. **浏览器兼容**：推荐使用 Chrome/Edge 最新版本
4. **网络要求**：需要访问外部API和政府采购网站

## 📋 试用流程
1. **规则确诊**：输入URL和初始黑白名单，点击"规则确诊"
2. **DeepSeek推荐**：查看AI推荐的规则，编辑后保存
3. **启动生产**：配置参数后启动爬虫任务
4. **数据验证**：查看提取结果，提交反馈

## ❓ 常见问题
**Q: 启动时提示缺少依赖包？**
A: 手动安装：`pip install crawl4ai openai sqlalchemy fastapi uvicorn`

**Q: 无法访问 localhost:8000？**
A: 检查端口是否被占用，或防火墙设置

**Q: 前端界面显示但API调用失败？**
A: 检查后端日志，确保DeepSeek API密钥有效

**Q: 想停止服务？**
A: 在终端按 `Ctrl+C`

## 📞 技术支持
- 查看详细文档：`docs/` 目录
- API文档：启动后访问 http://localhost:8000/docs
- 后端日志：`logs/` 目录

## MaxClaw · 麦龙（云端）

**基本信息**
- 产品定位：MiniMax 提供的云端 AI Agent
- 部署方式：云端订阅制服务
- 接入方式：支持微信对话交互，具备用户长期记忆能力
- 已打通环境：已接入钉钉后台，可在钉钉群内被 @ 交互、监听群消息

**公开核心能力**

1. **云端持续运行** — 7×24 小时在线，不依赖本地电脑开机
2. **长期记忆与上下文理解** — 可记住历史对话、用户偏好与业务设定
3. **多平台协同交互** — 微信、钉钉等多入口联动
4. **内容生成与信息整合** — 文本撰写、总结提炼、方案梳理、知识问答
5. **任务调度与流程执行** — 规划任务步骤、调用联网信息、生成结构化结果
6. **团队协作** — 群聊中被动响应、自动汇总信息、提醒事项

**钉钉机器人实测状态（2026-04-05）**

| 功能 | 状态 |
|------|------|
| Stream 接收消息 | ✅ 已通 |
| 主动发消息到钉钉 | ✅ API 验证可行 |
| 群聊必须 @机器人 | ⚠️ 钉钉机制限制 |
| 语音转文字 | ✅ 钉钉自带，收到文字即可 |
| 录音卡群语音 | ⚠️ 能收，但需要 @她 |

## QClaw · 金龙（本地）

**基本信息**
- 产品定位：腾讯提供的本地端 AI Agent
- 部署方式：Mac 设备本地运行
- 扩展配置：用户已自行集成增强能力（非原生自带）

**公开原生能力**

1. **本地离线运行** — 模型与数据在本机执行，隐私性强
2. **子 Agent 体系** — 支持拆分子任务，多个子 Agent 协同
3. **本地文件与系统操作** — 直接读写本地文档、表格、文件夹
4. **轻量自动化与工具调用** — 打开应用、执行简单流程

**用户额外集成增强能力**

1. **代码能力增强** — 内置集成 Claude Code，具备专业级代码开发能力
2. **视觉感知能力** — 接入香港大学开源视觉项目，屏幕识别、界面理解
3. **系统自动化操作** — 模拟鼠标/键盘，桌面端流程自动化

## 任务A：安装心跳进程（一次性，5分钟）

心跳 plist 文件已经在仓库里，但从未安装过。现在执行：

```bash

ls ~/Desktop/jin-ai-workspace/tools/launchd/

# 安装
cp ~/Desktop/jin-ai-workspace/tools/launchd/com.jinai.claude-deputy.plist ~/Library/LaunchAgents/
cp ~/Desktop/jin-ai-workspace/tools/launchd/com.jinai.claude-nightly.plist ~/Library/LaunchAgents/

# 验证
```

## 任务B：浏览器操作的正确方式（关于港大视觉工具）

你不需要安装 OpenCUA，也不需要任何新工具。

**你 CodeBuddy 里已经有视觉自动化能力**，直接用它操作浏览器：

### 正确操作步骤（找 ccgp.gov.cn 列表页 URL）：

1. 用 CodeBuddy 的 computer-use / 视觉工具打开浏览器
2. 访问 `https://www.ccgp.gov.cn/`
3. 截屏 → 找到搜索框或"采购公告"入口
4. 在搜索框里输入关键词（如"消防设备"）
5. 点击搜索，等待结果页加载
6. 截屏 → 记录结果页的 URL（应类似 `https://search.ccgp.gov.cn/bxsearch?...`）

**不需要任何第三方模型，不需要 GPU，CodeBuddy 自带的能力足够完成这个操作。**

如果操作过程中遇到具体报错，把错误信息写入 `briefs/t013-report.md` 的"问题记录"里，不要上报给 Boss。

## 第三步：视觉导航找列表页 URL

用你的视觉 AI 能力（港大项目）模拟人操作浏览器，找到以下入口：

**【场景A】ccgp.gov.cn — 按设备品类搜索，拿关键词列表页URL**

这是全国通用采购平台，按设备类型搜索，不是搜公司名。

操作步骤：
1. 打开 https://www.ccgp.gov.cn/
2. 找到公告搜索入口（搜索框）
3. 分别输入以下关键词，每个都执行一次搜索：
   - "消防设备"
   - "机电设备"
   - "楼宇自控"
4. 记录每个关键词对应的结果列表页 URL（格式类似 `https://search.ccgp.gov.cn/bxsearch?kw=消防设备&...`）
5. 这三个 URL 就是后续爬取的入口

**产出**：3 个搜索结果列表页 URL，写入 `briefs/t013-report.md`

---

**【场景B】成都建工集团 或 成都华西集团 — 找该单位自己的招标公告入口**

这是特定目标客户监控，要找的是该单位**自己发布**招标公告的页面，不是在 ccgp 搜他们名字。

操作步骤：
1. 用搜索引擎找到成都建工集团（或成都华西集团）的**官方网站**
2. 在官网内导航，找到"招标采购"/"供应商"/"采购公告"入口
3. 进入后找到公告列表页
4. 记录该列表页 URL

注册情况处理：
- 无门槛注册 → 自行完成注册
- 需扫码/人工审核 → 停下，在报告里写"需要 Boss 协助注册"，记录注册页面 URL

**产出**：该单位采购公告列表页 URL（找到几个写几个），写入 `briefs/t013-report.md`

## 第五步：第一层筛选（快速，批量一次完成）

**规则设计原则**：
- 用一次 AI 调用处理所有标题，不逐条调用
- 分三类：白名单（直接入队）/ 黑名单（直接丢弃）/ 灰名单（入队待二层判断）

**黑名单关键词（命中即丢弃，不进队列）**：
- 施工、土建、建筑施工、装修、装饰、园林绿化、道路、拆除、脚手架

**白名单关键词（命中直接入队，标记为"确认相关"）**：
- 消防设备、机电设备、楼控、楼宇自控、暖通、弱电、电气设备、仪表、自动化设备、配电

**灰名单（其他所有条目）**：
- 不确定的一律入队，标记为"待二层判断"，由麦龙打开详情页后决定

**输出格式**：
```json
[
  {"title": "...", "detail_url": "...", "source": "ccgp", "layer1": "confirmed"},
  {"title": "...", "detail_url": "...", "source": "ccgp", "layer1": "pending"},
]
```
黑名单条目不出现在文件里。

**输出文件**：
- `intel/queue-ccgp.json`
- `intel/queue-成都建工.json`（如有）

# 金龙能力摸底报告

> 回答日期：2026-04-05 | 回答者：金龙

---

## 一、桌面操作能力

1. **直接操作 macOS 桌面？**
   - **部分能** - 我可以通过 exec 命令执行终端命令、用 `open` 命令打开应用、用 `osascript` 操作部分 GUI 元素。但不能直接模拟鼠标点击或键盘输入（需要通过终端命令间接实现）。

2. **处理中文界面？**
   - **能** - 我能读取中文文件名、处理中文文本内容。终端命令中的中文显示可能有编码问题，但基本可以处理。

3. **多窗口操作？**
   - **部分能** - 我没有直接管理多个窗口的能力，但可以通过终端命令（如 osascript）间接切换窗口焦点。

---

## 二、文件操作能力

4. **下载文件？**
   - **能** - 我可以用 `curl`、`wget`、`yt-dlp`、`you-get` 等命令下载文件。

5. **处理压缩包？**
   - **能** - 我可以用 `tar`、`zip`、`unzip`、`7z` 等命令解压或打包。

6. **文件格式转换？**
   - **部分能** - 我可以通过调用外部工具进行转换（如 ffmpeg 视频转码、pandoc 文档转换），但本身没有内置转换能力。

7. **批量重命名？**
   - **能** - 我可以用脚本（bash/python）批量重命名文件。

---

## 三、网络与通信能力

8. **访问需要登录的网站？**
   - **不能** - 我没有内置浏览器自动化能力（需要额外配置 browser 工具），无法直接登录网站。

9. **处理验证码？**
   - **不能** - 我无法识别或处理验证码图片。

10. **监听消息推送？**
    - **能** - 我可以通过配置 heartbeat（心跳）或 cron（定时任务）来定时检查新消息。

---

## 四、多媒体处理能力

11. **下载视频？**
    - **能** - 我可以用 `yt-dlp`、`you-get` 等工具下载 B站、YouTube 等视频。

12. **处理视频文件？**
    - **部分能** - 我可以调用 `ffmpeg` 进行剪辑、压缩、提取音频等操作。

13. **生成图片？**
    - **能** - 我可以调用 DALL-E、Stable Diffusion 等 API 生成图片（需要配置 API）。

14. **语音转文字？**
    - **能** - 我可以用 Whisper 工具进行音频/视频转写。

---

## 五、自动化与集成能力

15. **设置定时任务？**
    - **能** - 我可以用 cron 工具设置定时任务。

16. **调用外部 API？**
    - **能** - 我可以用 `curl` 命令调用各种 REST API。

17. **操作数据库？**
    - **能** - 我可以通过命令行工具（mysql、psql、mongosh）连接并操作数据库。

---

## 六、钉钉连接器专项

18. **配置 Gateway 的 channel？**
    - **能** - 我可以用 `gateway config.patch` 或 `gateway config.apply` 命令修改配置。

19. **重启 Gateway 服务？**
    - **能** - 我可以用 `gateway restart` 命令或 `kill -9` 进程来重启。

20. **建立钉钉 Stream 长连接？**
    - **能** - 我可以配置 dingtalk-connector 插件，插件会使用钉钉 Stream SDK 建立 WebSocket 长连接。

21. **收发钉钉机器人消息？**
    - **能** - dingtalk-connector 插件支持接收和发送钉钉机器人消息。

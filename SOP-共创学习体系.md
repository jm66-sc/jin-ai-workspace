## 每日运行流程

### 麦龙每天做什么

**第一步：采集（每天 09:00 前完成）**
- 从指定监控源（孟健/陈巍/AI超元域/掘金/博客园/GitHub）读取内容
- 每天约 **10 篇**（不追求多，追求有价值）
- 视频 → 提取字幕/讲解内容 → 转为文字稿

**第二步：存档**
- 原文存入：`共创资料池/📥 原文存档/YYYY-MM/序号-来源-标题.md`
- 同时生成中心思想卡片

**第三步：生成中心思想卡片**
- 存入：`共创资料池/🧩 中心思想卡片/YYYY-MM/卡片-序号-一句话核心.md`
- 格式：
  ```
  # 卡片 - 孟健：如何让AI站前端不像AI
  
  **来源**：知乎专栏 | **日期**：2026-04-03
  **中心思想**：用 AI Agent 接管前端全流程，做出来的网站要像人做的而不像 AI 做的。
  
  **对我有用的点**：
  - [ ] 原文的核心操作步骤（简述）
  - [ ] 对我们工作可能的帮助（简述）
  - [ ] 我的感受/疑问（留给 Boss 读后补充）
  ```

**第四步：推送**
- 微信告诉 Boss："今天 10 篇已存档，随时可以说'看今天的卡片'来调取"
- 不强制 Boss 立即读，有空再说

---

### Boss 随时做什么

**方式一：随时调取**
> "麦龙，把今天的卡片调出来看看"

**方式二：一篇一篇推**
> "给我推一篇"

麦龙推出一篇，Boss 阅读后口头评论，麦龙记录到笔记文件。

**方式三：集中看**
> "把本周的卡片全给我看看"

---

### Boss 评论后，麦龙做什么

Boss 口头评论 → 麦龙记录进阅读笔记：

```
# 笔记 - Boss读后感：孟健-如何让AI站前端不像AI

**日期**：2026-04-07
**Boss 感受**：
- [记录 Boss 的原话/感受]

**麦龙备注**：
- [补充我的理解]

**是否落地**：待讨论 / 已落地 / 暂不适用
```

## 某天讨论时做什么

当 Boss 说"我们来讨论一下最近学到的东西"：

1. 调出这个月的共创讨论记录
2. 对齐认知——大家都读过这些资料，有共同语言
3. 讨论某个想法是否可以用这段时间学到的东西来落地
4. 形成共识后 → 写入 TASKS.md → 执行

## 核心原则

1. **原文必须存档**——找到什么存什么，方便以后回溯
2. **中心思想必须生成**——读完了才真正理解
3. **感受必须记录**——Boss 的感受是共创的核心，不能丢失
4. **不强求立即落地**——先积累，等共鸣到了再动手
5. **共同记忆池**——麦龙和金龙都读过这个池子，认知对齐后讨论才有意义
